# Business Expansion Screen (Ekspansi & Pengembangan)

This screen simulates regional feasibility studies and workflows for starting new cooperative business units (e.g. fertilizer supply, clean water).

---

### 📋 Feature Workflow (Simulation Stepper)
1. **Pengumpulan Data**: Loads cooperative membership levels, transaction summaries, and demographic targets.
2. **Analisis Regional**: Consults regional commodity maps and district supply statistics.
3. **Skrining Kelayakan**: Calculates NPV, IRR, and Benefit-Cost (B/C) indicators.
4. **RAT Presentation**: Generates printable PDF briefs for member voting assemblies.

---

### 💻 Developer Sandbox Details
- Clicking **Mulai Analisis Potensi** triggers a simulated step-by-step console print log.
- Feeds local diagnostics state into active React render elements.
- Simulates an AI-based recommendation that opens a detailed fertilizer supply unit preview card.
