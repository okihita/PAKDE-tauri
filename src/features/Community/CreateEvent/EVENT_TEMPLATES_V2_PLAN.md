# Event Templates v2 — Engagement-Driven Events

Design plan for expanding the event template system with **fun day events** that drive
member retention, foot traffic to the cooperative store, and community engagement. Each template
includes a **"Why" rationale** (cooperative manager guidance), actionable **checklist**, and
**quest system integration** hooks.

---

## Philosophy: Events as Cooperative Growth Engines

A village cooperative isn't just a shop — it's the social nucleus of the community. Events are
the most effective tool for:
- **Member retention** — members who attend events renew savings, take loans, shop at the waserda
- **New member acquisition** — visitors become prospects
- **Brand reinforcement** — "Koperasi Desa" becomes synonymous with community value
- **Revenue spikes** — event days create foot traffic → impulse purchases at the cooperative store

Every event template is designed to answer three questions for the manager:
1. **Why?** — what problem does this solve, what opportunity does it capture?
2. **How?** — step-by-step checklist with concrete deadlines
3. **What's the ROI?** — measurable outcomes linked to cooperative health metrics

## Quest System Integration

Each event checklist item can generate a **quest** on the manager's homepage. The quest
engine hooks into the checklist deadlines:

```
┌──────────────────────────────────────────────────────┐
│               HOMEPAGE QUEST SYSTEM                  │
├──────────────────────────────────────────────────────┤
│                                                      │
│  📋 TODAY (D-7 until Tournament)                     │
│  ┌────────────────────────────────────────────────┐  │
│  │  ⚽  Print tournament bracket and confirm        │  │
│  │  referees                                       │  │
│  │  ⏱  15 min  │  📍 Event Page → Checklist       │  │
│  │  [Mark Complete]                                │  │
│  └────────────────────────────────────────────────┘  │
│                                                      │
│  📋 THIS WEEK (Tournament Prep)                      │
│  ┌────────────────────────────────────────────────┐  │
│  │  📋  Register 8 teams for futsal tournament     │  │
│  │  ⏱  30 min  │  📍 Event Page → Registration    │  │
│  │  [Mark Complete]  ████████░░  4/8 teams         │  │
│  └────────────────────────────────────────────────┘  │
│                                                      │
│  🏆 MILESTONE                                        │
│  ┌────────────────────────────────────────────────┐  │
│  │  🎯  Successfully run 3 cooperative events       │  │
│  │  ██░░░░░░░░  1/3 events                         │  │
│  │  Reward: Unlock "Event Organizer" badge, +500 XP │  │
│  └────────────────────────────────────────────────┘  │
│                                                      │
└──────────────────────────────────────────────────────┘
```

**Quest Generation Rules:**
1. When an event is created from a template, all checklist items are scheduled as quests
2. Quest deadlines = checklist D-n dates computed from event date
3. Daily quests: items due within 3 days of today
4. Weekly quests: items due within 7 days
5. Milestone quest: cumulative — "organize 3 events" → unlocks badge at level 3 Business Units

**Badge Rewards:**
- 🏅 **Event Organizer** — 3 events completed → +500 XP (Business Units aspect)
- 🏅 **Community Pillar** — 10 events completed → +2000 XP, cooperative rank visibility boost
- 🏅 **Village Fest Host** — 1 large event (300+ attendees) → +1000 XP
